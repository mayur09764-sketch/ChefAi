
# 🍴 ChefAI — Smart Kitchen Assistant

AI-powered recipe and meal planning web app


## Features
- 🤖 AI Recipe Generator (Gemini 2.5 Flash)
- 📅 Weekly Meal Planner
- 🛒 Shopping List
- 🔐 Login / Signup

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: MySQL
- AI: Google Gemini 2.5 Flash
